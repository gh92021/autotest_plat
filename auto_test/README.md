自动化测试平台
API测试
基于 Django + pytest 构建的 API 自动化测试平台，支持接口管理、测试用例编排、参数化测试、性能测试等功能。

功能特性
接口管理：支持 RESTful API 的增删改查，支持导入 cURL
测试用例管理：可视化编排测试步骤，支持前置/后置脚本
参数化测试：支持多组参数批量执行测试用例，支持用例变量设置
测试套件：组合多个测试用例形成测试计划
执行引擎：基于 pytest 实现，支持 Allure 报告生成
全局变量：支持测试环境管理和全局参数配置

技术栈
分类	技术
后端框架	Django
任务队列	Celery
测试框架	pytest
报告生成	Allure
数据库	SQLite/MySQL

使用说明
1. 接口管理
支持手动录入或导入 cURL
每个接口关联到指定环境
2. 测试用例创建
创建用例，填写基本信息
添加测试步骤，关联接口，配置请求参数和断言
可选配置参数化数据进行批量测试
3. 执行测试
单用例执行：
支持单步骤调试和完整用例执行
套件执行：
创建测试计划，关联多个测试用例
执行测试计划，系统自动异步执行所有用例
4. 查看报告
测试用例或计划执行完成后自动生成测试报告
支持 Allure 可视化报告查看
5. 定时任务
全量测试用例定时巡检，重复失败用例自动禁用
定时清理过期执行历史记录

性能测试
基于 Django + Locust 构建的性能测试平台，支持测试脚本管理、测试任务配置、测试执行和结果分析功能。

功能特性
脚本管理: 支持 AI 生成脚本和人工编写脚本；支持 AI 生成测试数据
测试任务配置: 配置并发用户数、启动速率、运行时长等参数
单机/分布式运行: 支持单机和分布式压测模式
实时监控: 实时采集和展示测试指标（RPS、响应时间、失败率等）
结果分析: 生成测试报告，AI 分析及提供优化建议
